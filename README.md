# Vagrant Datacenter

A repository used for offline local testing and development.

## Commands

Bring up all nodes
`vagrant up`

SSH into a node
`vagrant ssh consul-x`

Bring up an individual node
`vagrant up consul-1`
`vagrant up consul-2`
`vagrant up consul-3`

Destroy an individual node
`vagrant destroy consul-x`

Destroy all nodes
`vagrant destroy -f`
